
User authentication: How would secure your API if you were requested to do so?

Η πρώτη επιλογή στη διασφάλιση του REST API θα ήταν η χρήση https και στη 
συνέχεια θα ήταν ένα API «κλειδί» για να ελέγχω το πλήθος των κλήσεων που 
θα λαμβάνει το api. Επίσης θα βοηθούσε στο να αποτραπεί μία ανώνυμη είσοδος. 
Μία εναλλακτική μορφή διασφάλισης του REST API θα ήταν η χρήση των «tokens» μέσω 
των οποίων αποφεύγεται η αποθήκευση πληροφοριών στο server. 
Επιπλέον δικλείδα ασφαλείας αποτελεί η χρήση «tokens» που θα λήγουν ανά ένα 
συγκεκριμένο μικρό χρονικό διάστημα (π.χ. 1-2 ημέρες).
